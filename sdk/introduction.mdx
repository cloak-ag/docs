Cloak SDK exposes two interfaces, with UTXO as the primary integration path:

- **Note API** via `CloakSDK` (`deposit`, `withdraw`, `send`, `swap`)
- **UTXO API** via top-level functions (`transact`, `transfer`, `swapUtxo`, ...)

Most production integrations use the UTXO API (`/transact`, `/transact_swap`).


```bash yarn
yarn add @cloak.ag/sdk @solana/web3.js
```

```bash pnpm
pnpm add @cloak.ag/sdk @solana/web3.js
```

```bash bun
bun add @cloak.ag/sdk @solana/web3.js
```


## Minimal note-based example

<CodeGroup>
```typescript Keypair bytes
import { CloakSDK } from "@cloak.ag/sdk";
import { Connection, Keypair } from "@solana/web3.js";

const connection = new Connection("https://api.mainnet-beta.solana.com", "confirmed");
const keypair = Keypair.fromSecretKey(/* Uint8Array secret key */);

const sdk = new CloakSDK({
  keypairBytes: keypair.secretKey,
  network: "mainnet",
  programId: new PublicKey("c1oak6tetxYnNfvXKFkpn1d98FxtK7B68vBQLYQpWKp"),
});

const deposit = await sdk.deposit(connection, 100_000_000);
const recipient = Keypair.generate().publicKey;

const sent = await sdk.send(connection, deposit.note, [
  { recipient, amount: 30_000_000 },
]);

console.log("Send tx:", sent.signature);
```

```typescript Wallet adapter
import { useMemo } from "react";
import { CloakSDK } from "@cloak.ag/sdk";
import { useWallet } from "@solana/wallet-adapter-react";
import { PublicKey, Connection, Keypair } from "@solana/web3.js";

export function useCloakNoteClient() {
  const { publicKey, signTransaction, sendTransaction } = useWallet();

  return useMemo(() => {
    if (!publicKey || !signTransaction || !sendTransaction) return null;

    return new CloakSDK({
      wallet: { publicKey, signTransaction, sendTransaction },
      network: "mainnet",
      programId: new PublicKey("c1oak6tetxYnNfvXKFkpn1d98FxtK7B68vBQLYQpWKp"),
    });
  }, [publicKey, signTransaction, sendTransaction]);
}

async function exampleFlow(sdk: CloakSDK, connection: Connection) {
  const deposit = await sdk.deposit(connection, 100_000_000);
  const recipient = Keypair.generate().publicKey;
  await sdk.send(connection, deposit.note, [{ recipient, amount: 30_000_000 }]);
}
```
</CodeGroup>

## Core protocol fees

Shared protocol constants:

- Fixed: `5_000_000` lamports (0.005 SOL)
- Variable: `amount * 3 / 1000` (0.3%)
- Minimum deposit: `10_000_000` lamports (0.01 SOL)
