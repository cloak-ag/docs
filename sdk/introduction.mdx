## Quick example

```typescript
import { CloakSDK, getDistributableAmount } from "@cloak.ag/sdk";
import { Connection, Keypair, PublicKey } from "@solana/web3.js";

// Initialize
const sdk = new CloakSDK({
  keypairBytes: keypair.secretKey, // or wallet adapter
  network: "devnet",
});

// Deposit SOL
const { note } = await sdk.deposit(connection, 100_000_000);

// Withdraw privately
await sdk.withdraw(connection, note, recipient, { withdrawAll: true });
```

</Tip>

## Fee structure

| Fee type | Amount | Description |
|----------|--------|-------------|
| Fixed fee | 0.005 SOL | Base fee per withdrawal |
| Variable fee | 0.3% | Percentage of deposit amount |

```typescript
import { getDistributableAmount, calculateFee } from "@cloak.ag/sdk";

const deposited = 100_000_000; // 0.1 SOL
const fee = calculateFee(deposited);           // 5,300,000 lamports
const afterFees = getDistributableAmount(deposited); // 94,700,000 lamports
```

## Security model

<Warning>
**Protect your notes!** A Cloak Note contains spending secrets. Anyone with the note can withdraw the funds. Treat notes like private keys.
</Warning>

| Guarantee | Description |
|-----------|-------------|
| **Transaction privacy** | On-chain observers cannot link deposits to withdrawals |
| **Amount privacy** | Individual amounts hidden in the anonymity set |
| **Double-spend protection** | Nullifiers prevent reusing notes |
| **Proof verification** | All withdrawals require valid ZK proofs |
