This quickstart uses the current UTXO API (`transact` + `partialWithdraw`).

## Prerequisites

- Node.js 18+
- A funded Solana keypair
- Relay URL (hosted endpoint)
- Solana RPC URL
- Hosted circuit base URL: `https://www.cloak.ag/circuits`

## Install

```bash
mkdir cloak-demo && cd cloak-demo
npm init -y
npm install @cloak.ag/sdk @solana/web3.js tsx
```

## Example script

<CodeGroup>
```typescript Keypair bytes
import {
  createUtxo,
  createZeroUtxo,
  generateUtxoKeypair,
  partialWithdraw,
  transact,
  setCircuitsPath,
} from "@cloak.ag/sdk";
import { Connection, Keypair, PublicKey, LAMPORTS_PER_SOL } from "@solana/web3.js";
import { readFileSync } from "node:fs";
import { join } from "node:path";

const connection = new Connection(
  process.env.SOLANA_RPC_URL || "https://api.mainnet-beta.solana.com",
  "confirmed"
);

const keypairPath = process.env.KEYPAIR_PATH || join(process.env.HOME!, ".config/solana/id.json");
const keypair = Keypair.fromSecretKey(
  Uint8Array.from(JSON.parse(readFileSync(keypairPath, "utf8")))
);

const programId = new PublicKey(
  process.env.CLOAK_PROGRAM_ID || "c1oak6tetxYnNfvXKFkpn1d98FxtK7B68vBQLYQpWKp"
);
const relayUrl = process.env.CLOAK_RELAY_URL || "https://your-relay.example.com";
setCircuitsPath(process.env.CIRCUITS_PATH || "https://www.cloak.ag/circuits");

const owner = await generateUtxoKeypair();
const depositAmount = BigInt(0.02 * LAMPORTS_PER_SOL); // 0.02 SOL

const depositUtxo = await createUtxo(depositAmount, owner);
const deposit = await transact(
  {
    inputUtxos: [await createZeroUtxo()],
    outputUtxos: [depositUtxo],
    externalAmount: depositAmount,
    depositor: keypair.publicKey,
  },
  {
    connection,
    programId,
    relayUrl,
    depositorKeypair: keypair,
  }
);

const recipient = Keypair.generate().publicKey;
const withdraw = await partialWithdraw(
  deposit.outputUtxos,
  recipient,
  BigInt(0.005 * LAMPORTS_PER_SOL),
  {
    connection,
    programId,
    relayUrl,
    depositorKeypair: keypair,
    cachedMerkleTree: deposit.merkleTree,
  }
);

console.log("Deposit tx:", deposit.signature);
console.log("Withdraw tx:", withdraw.signature);
```

```typescript Wallet adapter
import { useCallback } from "react";
import {
  createUtxo,
  createZeroUtxo,
  generateUtxoKeypair,
  partialWithdraw,
  transact,
  setCircuitsPath,
} from "@cloak.ag/sdk";
import { Connection, PublicKey, LAMPORTS_PER_SOL, Keypair } from "@solana/web3.js";
import { useWallet } from "@solana/wallet-adapter-react";

const connection = new Connection(process.env.NEXT_PUBLIC_RPC_URL!, "confirmed");
const programId = new PublicKey("c1oak6tetxYnNfvXKFkpn1d98FxtK7B68vBQLYQpWKp");
const relayUrl = process.env.NEXT_PUBLIC_RELAY_URL!;
setCircuitsPath("https://www.cloak.ag/circuits");

export function useQuickstartFlow() {
  const { publicKey, signTransaction } = useWallet();

  return useCallback(async () => {
    if (!publicKey || !signTransaction) throw new Error("Wallet not ready");

    const owner = await generateUtxoKeypair();
    const depositAmount = BigInt(0.02 * LAMPORTS_PER_SOL);
    const depositUtxo = await createUtxo(depositAmount, owner);

    const deposit = await transact(
      {
        inputUtxos: [await createZeroUtxo()],
        outputUtxos: [depositUtxo],
        externalAmount: depositAmount,
        depositor: publicKey,
      },
      {
        connection,
        programId,
        relayUrl,
        signTransaction,
        depositorPublicKey: publicKey,
        walletPublicKey: publicKey,
      }
    );

    const recipient = Keypair.generate().publicKey;
    const withdraw = await partialWithdraw(deposit.outputUtxos, recipient, BigInt(0.005 * LAMPORTS_PER_SOL), {
      connection,
      programId,
      relayUrl,
      walletPublicKey: publicKey,
      cachedMerkleTree: deposit.merkleTree,
    });

    return { deposit, withdraw };
  }, [publicKey, signTransaction]);
}
```
</CodeGroup>

Note: `https://www.cloak.ag/circuits` is a base path (directory listing is not enabled).

SDK proof generation needs concrete files under that path, including:

- `https://www.cloak.ag/circuits/withdraw_regular_js/withdraw_regular.wasm`
- `https://www.cloak.ag/circuits/withdraw_regular_final.zkey`
- `https://www.cloak.ag/circuits/withdraw_swap_js/withdraw_swap.wasm` (swap flows)

Run it (keypair script):

```bash
npx tsx demo.ts
```

## Important runtime notes

- Relay validates `proof_bytes` as 256 bytes and `public_inputs` as 264 bytes.
- SDK retries root-staleness (`0x1001`) by rebuilding proofs.
- Viewing-key registration is enforced by default in SDK transaction flows.
- If your deployment uses a different program, pass explicit `programId`.

## Verify scanner output

Add a scanner step in your app/script:

<CodeGroup>
```typescript Keypair bytes
import { scanTransactions, toComplianceReport } from "@cloak.ag/sdk";

const scan = await scanTransactions({
  connection,
  programId,
  viewingKeyNk: /* your 32-byte nk */,
});

const report = toComplianceReport(scan);
console.log(report.summary);
```

```typescript Wallet adapter
import { scanTransactions, toComplianceReport } from "@cloak.ag/sdk";

const nk = await getChainNoteNkFromWalletSession(); // 32-byte nk from your wallet key flow

const scan = await scanTransactions({
  connection,
  programId,
  viewingKeyNk: nk,
});

console.log(toComplianceReport(scan).summary);
```
</CodeGroup>

This returns chain-native totals (deposits, withdrawals, fees, final balance).
